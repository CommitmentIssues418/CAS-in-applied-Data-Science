# Module 2: Statistical Inference for Data Science

## Module Description
This module reviews and deepens fundamental statistical concepts to prepare participants for data analysis and modeling.  
The focus lies on understanding distribution properties, confidence intervals, hypothesis testing, and exploratory statistical evaluation.

---

## Learning Objective:
recall or become familiar with typical statistical concepts for describing and analysing data.

---

## Contents
- Descriptive statistics  
- Probability distributions  
- Estimators and sampling  
- t-tests, ANOVA, chi-square tests  
- Assumptions & diagnostic plots  
- Correlation and covariance
