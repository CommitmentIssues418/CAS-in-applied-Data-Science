# Module 1: Data Acquisition and Management

## Module Description
Module 1 introduces the foundations of **data acquisition**, **data formats**, and **data management planning**.  
Participants learn to understand different types of data sources, define data strategies, and structure metadata and documentation workflows.

---

## Learning Objectives
learn to understand different data sources and types and how to design data management models and plans.


---

## Contents
- Data formats: CSV, JSON, XML, Parquet  
- APIs and data collection  
- Data storage concepts  
- Data lifecycle management  
- Metadata, ontologies, documentation  
- FAIR and reproducibility principles
